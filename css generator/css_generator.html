CSS Generater helps you to create css code for web developers. Browser engine use css property to format html elements.

There are three ways to define CSS property
1) Include external *.css file inside <header> element
2) define css inside <style> element
3) Use inline style attribute

inline style is always in first priority given by browser engine

Code
external style sheet
<link href="style.css" rel="stylesheet" type="text/css">

internal style
<style>
div {
background-color:red;
}
</style>

inline style
<div style="background-color:red" > </div>
